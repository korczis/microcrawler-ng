# microcrawler-ng

Microcrawler

## Examples

### google

`microcrawler -t google "https://google.com/search?hl=en&q=korczis"`

### hackernews

`microcrawler -t hackernews https://news.ycombinator.com`

### xkcd

`microcrawler -t xkcd http://xkcd.com/`

### youjizz

`microcrawler -t youjizz http://youjizz.com`
